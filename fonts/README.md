# Zine Fonts

This directory contains a curated selection of open-source fonts perfect for zine creation. All fonts are licensed under the SIL Open Font License (OFL) or Apache License 2.0.

## Available Fonts

### Typewriter & Retro
- **Special Elite** - Classic typewriter font, perfect for vintage zines
- **Press Start 2P** - 8-bit pixel font for retro gaming aesthetics
- **VT323** - Monospace terminal font with vintage computer vibes

### Handwriting & Script
- **Caveat** - Casual handwriting style
- **Indie Flower** - Friendly handwritten font
- **Amatic SC** - Casual handwriting with bold weight
- **Satisfy** - Elegant script/cursive font

### Display & Decorative
- **Permanent Marker** - Bold hand-drawn marker style
- **Bangers** - Comic book style font
- **Creepster** - Horror/spooky display font

### Modern Monospace
- **Roboto Mono** - Clean, modern monospace font

### Letraset-Style Geometric
- **Bebas Neue** - Condensed geometric sans-serif, classic transfer lettering aesthetic
- **Orbitron** - Geometric futuristic font with 1970s sci-fi influence
- **Audiowide** - Retro technology styled display font
- **Russo One** - Bold geometric sans-serif
- **Righteous** - 1970s inspired display font
- **Michroma** - Geometric sans-serif with technical feel
